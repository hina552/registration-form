# Registration Form – Clubs & Activities

A simple registration form for enrolling students in clubs and extracurricular activities.

This project was built independently as practice while following **The Odin Project** curriculum, inspired by a school-style assignment.

## Features

- Student registration form  
- Gender, field, and group selection  
- Club selection  
- Registration list and statistics  
- Responsive layout  

## Built With

- HTML  
- CSS  
- Vanilla JavaScript  
- Font-Awesome

## What I Practiced

- Semantic HTML structure  
- Form layout and styling  
- Flexbox-based layouts  
- DOM manipulation  
- Basic state management with JavaScript
